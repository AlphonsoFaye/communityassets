# Four Strand Braid

* Author: brkurt
* Category: Hair
* Compatibility: 1.1.x
* License: CC0

Needs refinement, but it's good enough to use as is. This is a four-strand braid that uses two rules for braiding: one for the outer strands, one for the inner strands. Should work for any female model.
